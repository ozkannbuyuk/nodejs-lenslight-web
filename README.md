❓ About
</h2>

I coded a photo-sharing website with Node.js:
- ✔ &nbsp; Member login and registration processes
- ✔ &nbsp; Photo sharing
- ✔ &nbsp; Sending emails

Install

To install the project, navigate to the directory and run:

- `npm i`

Run

To run the project, run the following command:

- `npm start`
